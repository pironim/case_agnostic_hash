# case_agnostic_hash
Hash with Case Insensitive access to fields

## Installation

```ruby
gem 'case_agnostic_hash'
```

And then execute:

    $ bundle install

## Access your keys formatted in Another World
### Available Cases
- snake_case
- PascalCase
- shakeCase
- kebab-case
- SCREAMING_SNAKE_CASE
- Camel_Snake_Case
- MalFOrmed-Case
